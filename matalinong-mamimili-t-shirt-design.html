24. A factory makes T-shirts in 5 colors, 4 design and 3 sizes how many different T-shirts can be made? ​


Answer:

60 different T-shirt

Step-by-step explanation:

Following the Fundamental Principle of Counting. If an event has m possible outcomes, and when it has been done, a second event has n possible outcomes, and when it has been done, a third event has p possible outcome, and so forth, then the total number of outcomes is the product of m, n, P, and so forth.

That is mxnxp...

5 colors x 4 designs x 3 sizes = 60 different T-shirt.

Hope it helps. Thank you.
