3. Impact to the society of Luz Oliveros-Belardo's contribution​


The impact of Luz Oliveros-Belardo to the society can be associated to her discovery of various essential oils for medical use. With her expertise in Phytochemical research, the discovered extracts from different plant species are known today to treat wide array of diseases and health issues. In fact, her discovery paved way to the use of natural plant extracts as components of drugs we use in the society in the present. Not just in pharmaceuticals, essential oils were now used as ingredients in production of cosmetics and aerosols. Thus, Belardo’s discovery made huge impact in the health and cosmetics industry.

Luz Oliveros-Belardo’s Career

Luz Belardo was specialized in Pharmaceutical Chemistry having earned a Bachelor’s degree in Pharmacy, Masters in Pharmaceutical Chemistry, and PhD in Pharmacy. Luz Oliveros-Belardo’s Awards

Therefore, to find the total cost of 5 reams of bond paper, you need to multiply the cost of one ream by the number of reams you want to purchase. In this case, the cost of one ream is x-4 pesos, and you want to purchase 5 reams, so the total cost will be 5x-20 pesos.

Which function best represents the total cost C if you are to buy x reams of long bond paper?​


Answer:

Piecewise Function:

              { 200x   if x < 50

       C = { 185x    if 50 ≤ x < 75

              { 160x   if x ≥ 75

Step-by-step explanation:

WHICH FUNCTION BEST REPRESENTS?Piecewise functions are used when a function requires more than one formula to produce the desired result.This describes when an input value crosses specific "boundaries," causing a rule or relationship to change.

In our given scenario, we have three boundaries:

For orders of less than 50, the bookstore charges 200php each.For orders of 50 or more but less than 75, each ream costs 185php.For orders of 75 and above, each ream costs 160php.

Before creating the function, one should first recall certain symbols for inequalities.

Greater than ( > )Greater than or equal to ( ≥ )Less than ( < )Less than or equal to ( ≤ )

We can now translate each condition:

Let x be the number of reams of long bond paper.

1.) For orders of less than 50, the bookstore charges 200php.

Translation: If x < 50, then each x is multiplied by 200.

2). For orders of 50 but less than 75, each ream costs 185php.

Translation: If 50 < x < 75, then each x is multiplied by 185.

Read as “if x is greater than or equal to 50 but less than 75”

3.) For orders of 75 and above, each ream costs 160php.

Translation: If x > 75, then each x is multiplied by 160.

Finalize our function:

      { 200x   if x < 50

C = { 185x    if 50 ≤ x < 75

      { 160x   if x ≥ 75
