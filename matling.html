a. To find x, we can set the expressions for LE and IT equal to each other since they both represent the length of the same side of the rectangle:

LE = IT

x + 5 = 2x - 10

Adding 10 to both sides:

x + 15 = 2x

Subtracting x from both sides:

15 = x

So LE = x + 5 = 20, and IT = 2x - 10 = 20.

b. To find x, we can use the fact that the sum of the angles in a rectangle is 360 degrees.

m∠ILE + m∠LIT = 180 degrees (since ILE and LIT are adjacent angles)

m∠ILE + m∠LIT + m∠MAT + m∠LEA = 360 degrees (since MAT and LEA are also adjacent angles)

Substituting the expressions for m∠ILE and m∠LIT:

(3x - 12) + (x + 10) + m∠MAT + 90 = 360

Simplifying and solving for x:

4x - 2 = 260

4x = 262

x = 65.5

Therefore, MATLE = 2x - 7 = 123.

a. Since LIET is a rhombus, all four angles are equal. Let's call this common angle "y".

m∠LTE = 4x - 30

m∠ET = x + 10

We can use the fact that the sum of the angles in a quadrilateral is 360 degrees to solve for y:

m∠LTE + m∠ET + y + y = 360

Substituting the expressions for m∠LTE and m∠ET:

(4x - 30) + (x + 10) + 2y = 360

Simplifying and solving for x:

5x - 20 = 350

5x = 370

x = 74

Substituting x back into the expressions for m∠LTE and m∠ET:

m∠LTE = 4(74) - 30 = 286

m∠ET = 74 + 10 = 84

Therefore, m∠EIT = m∠LEI = y = (360 - m∠LTE - m∠ET) / 2 = 245.

b. We are given that LE + LT = 30, and we know that LIET is a rhombus, so LE = LT.

Therefore,

LE + LE = 30

2LE = 30

LE = LT = 15

We are also given that PE = 7. Since LIET is a rhombus, PE is perpendicular to both LE and ET, so it divides LIET into two congruent triangles.

Let's call the point where PE intersects ET "F".

Since the two triangles are congruent, we know that EF = FT, and we can use the Pythagorean theorem to solve for TF:

TF² + FT² = TE²

(15 - 7)² + FT² = 15²

64 + FT² = 225

FT² = 161

FT ≈ 12.69

Therefore, T1 = LE + EF = 15 + 12.69 ≈ 27.69.

a.

In a rhombus, opposite angles are equal. So we have:

m angle LTE = m angle EIT

4x - 30 = x + 10

Solving for x, we get:

x = 10

Therefore, m angle EIT = m angle LEI = 20 degrees.

b.

In a rhombus, the diagonals are perpendicular bisectors of each other, so we have:

LE = IT = (1/2) * (LE + IT) = (1/2) * 30 = 15

Since LE + IT = 30 and PE = 7, we have:

T1 = LE - PE = 15 - 7 = 8.

Answer:

A. Magma in the mantle

Explanation:

Magma is the molten rock below the crust, in the mantle. Tremendous heat and pressure within the earth cause the hot magma to flow in convection currents. These currents cause the movement of the tectonic plates that make up the earth's crust.
