14. Researchers will conduct a survey to find out the opinion of residents in certain barangay on the ECQ implementation in their community. If there are 8,000 residents in the barangay and the researchers plan to use a sample using 10% margin of error, what should be the sample size? Pls use the Slovin’s Formula


[tex]{ \large{ \underline{ \overline{ \boxed{ \pink{\rm { \: SLOVINS\ FORMULA}}}}}}}[/tex]

Researchers will conduct a survey to find out the opinion of residents in certain  barangay on the ECQ implementation in their community. If there are 8,000 residents  in the barangay and the researchers plan to use a sample using 10% margin of error,  what should be the sample size?

[tex]\huge\purple{\overline{\quad\quad\quad\quad\quad\quad\quad\quad \ \ \ }}[/tex]

[tex]{ \large{ \underline{ \overline{ \boxed{ \pink{\rm { \: FORMULA}}}}}}}[/tex]

[tex]\rm{n= \frac{N}{1 + Ne^{2} } }[/tex]

n = sample size     N = population size e = margin of error

[tex]\huge\purple{\overline{\quad\quad\quad\quad\quad\quad\quad\quad \ \ \ }}[/tex]

[tex]{ \large{ \underline{ \overline{ \boxed{ \pink{\rm { \: SOLUTION}}}}}}}[/tex]

We know that n = 8,000 and e = 10% = 0.1, then the value of can be acquired from  the process below.

[tex]\huge\purple{\overline{\quad\quad\quad\quad\quad\quad\quad\quad \ \ \ }}[/tex]

[tex]\rm{n= \frac{N}{1 + Ne^{2} } } = \rm{\frac{8,000}{1+8,000(0.1)^{2} } } \rm{\ = 98.77\ or\ 99} }[/tex]

[tex]\huge\rm{\color{red}{So,\ the\ expected\ sample\ is\ 99.\ In\ symbol\ n = 99}}[/tex]

[tex]\huge\purple{\overline{\quad\quad\quad\quad\quad\quad\quad\quad \ \ \ }}[/tex]
