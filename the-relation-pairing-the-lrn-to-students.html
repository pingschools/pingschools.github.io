FUNCTIONA function is the relation between domain (x) and range (y) such that each value in the domain (x) corresponds to only one value in the range (y). Relations are said to be not functions if it violate this definition. They feature at least one value in the domain (x) that corresponds to two or more values in the range (y).

1. {(3,1),(4,2),(3,2),(1,2),(5,4)} - NOT FUNCTION
