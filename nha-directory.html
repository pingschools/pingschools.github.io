7. show directory buttons?


Answer:

Use is_dir() to see if item is a directory. Also, make sure that you actually mean $_POST, because that comes from the user, not from the file system. Elsewhere in your script, you use $file.

if(is_dir($_POST["folder_name"])) {

$output .=' ... folder html ... ';

} else {

$output .=' ... file html ... ';

}

Assuming that the user passes the folder name in $_POST, you might have something like this:

foreach(scandir($_POST['folder_name']) as $file)

{

if(is_dir($file)) {

} else {

$output .=' ... file html ... ';

18. What is Directory?Eplain..​

What directory means?

A directory is another name for a folder. File systems use directories to organize files within a storage device, such as an HDD or SSD.

For example, system files may be located in one directory, while user files may be stored in another. ... Each of these directories may contain files and other subdirectories.
