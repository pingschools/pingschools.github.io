2. *units of protractor in drawing a lot plan* three drafting practices in lot plotting​


Answer:

Units of protractor in drawing a lot plan

1. Full Circle contains 4 quadrants

2. One quadrant-90°(degrees)

3. One degree-60'(minutes)

Three drafting practices in lot plotting

1. All drawings and labels are inked

2. Bearing and Magnitude are centered to lot line and within the lot whenever possible

3. Lot No. and Block No. are written at the center of lot

NOTE: Bearing and Magnitude must be written and readable from South West quadrant

[tex]1,440m^{2}-144m^{2}=1,296m^{2}[/tex]

Each lot will be 1km long and the perimeter of each lot will be 4km.

Step by step explanation:

1. Convert the width of each lot from meters to kilometers.

40m = 0.04km

2. Convert the area of each lot from hectares to square kilometers.

5km² = 0.005km²

3. Divide the length of each lot by the width of each lot.

1km / 0.04km = 25

4. Multiply the answer from step 3 by the width of each lot.

25 * 0.04km = 1km

29. A triangle plot of land has sides a=3m,b=8 and 9m find the measures of angles of the triangular lot.​


LAW OF COSINES==============================

[tex] \large \bold{\blue{Problem:}} [/tex] A triangle plot of land has sides a=3m,b=8 and 9m find the measures of angles of the triangular lot.

[tex] \large \bold{\blue{Solution:}} [/tex] When three sides are given on a triangle, use the law of cosines to identify the other angles. We assigned variables to each angles, making ∆ABC. First is we solve for angle A, then angle B, and then angle C.

Solve for ∠A:

[tex] \sf cos \: A = \frac{b² \ + \ c² \ - a²}{2bc} [/tex]

[tex] \sf cos \: A = \frac{8² \ + \ 9² \ - 3²}{2(8)(9)} [/tex]

[tex] \sf cos \: A = \frac{64 \ + \ 81 \ - 9}{2(8)(9)} [/tex]

[tex] \sf cos \: A = \frac{136}{144} [/tex]

[tex] \sf \angle A = cos^{-1}(\frac{136}{144}) [/tex]

[tex] \bold{ \angle A = 19.19°} [/tex]

Solve for ∠B:

[tex] \sf cos \: B = \frac{a² \ + \ c² \ - b²}{2ac} [/tex][tex] \sf cos \: B = \frac{3² \ + \ 9² \ - 8²}{2(3)(9)} [/tex]

[tex] \sf cos \: B = \frac{9 \ + \ 81 \ - 64}{2(3)(9)} [/tex]

[tex] \sf cos \: B = \frac{26}{54} [/tex]

[tex] \sf \angle B = cos^{-1}(\frac{26}{54}) [/tex]

[tex] \bold{ \angle B = 61.22°} [/tex]

Solve for ∠C:

[tex] \sf \angle A + \angle B + \angle C = 180° [/tex]

[tex] \sf 19.19° + 61.22° + \angle C = 180° [/tex]

[tex] \sf 80.41° + \angle C = 180° [/tex]

[tex] \sf \angle C = 180° - 80.41° [/tex]

[tex] \bold{\angle C = 99.59°} [/tex]

30. the length of a rectangular lot is 10 m less than twice the width if the area of rectangular plot is 672 square meters what is the widtd?​


Answer:

The distance from x0 to x1 is:

S1 = √ (x1 − x0)2 + (y1 − y0)2

And let's use Δ (delta) to mean the difference between values, so it becomes:

S1 = √ (Δx1)2 + (Δy1)2

Now we just need lots more:

S2 = √(Δx2)2 + (Δy2)2

S3 = √(Δx3)2 + (Δy3)2

...

...

Sn = √(Δxn)2 + (Δyn)2

We can write all those many lines in just one line using a Sum:

S ≈

n

i=1

√(Δxi)2 + (Δyi)2

But we are still doomed to a large number of calculations!

Maybe we can make a big spreadsheet, or write a program to do the calculations ... but lets try something else.

We have a cunning plan:

have all the Δxi be the same so we can extract them from inside the square root

and then turn the sum into an integral.

Let's go:

First, divide and multiply Δyi by Δxi:

S ≈

n

i=1

√(Δxi)2 + (Δxi)2(Δyi/Δxi)2

Now factor out (Δxi)2:

S ≈

n

i=1

√(Δxi)2(1 + (Δyi/Δxi)2)

Take (Δxi)2 out of the square root:

S ≈

n

i=1

√1 + (Δyi/Δxi)2 Δxi

Now, as n approaches infinity (as we head towards an infinite number of slices, and each slice gets smaller) we get:

S =

lim

n→∞

n

i=1

√1 + (Δyi/Δxi)2 Δxi

We now have an integral and we write dx to mean the Δx slices are approaching zero in width (likewise for dy):

S =

b

a

√1+(dy/dx)2 dx

And dy/dx is the derivative of the function f(x), which can also be written f’(x):

S =

b

a

√1+(f’(x))2 dx

The Arc Length Formula

And now suddenly we are in a much better place, we don't need to add up lots of slices, we can calculate an exact answer (if we can solve the differential and integral).

Note: the integral also works with respect to y, useful if we happen to know x=g(y):

S =

d

c

√1+(g’(y))2 dy

So our steps are:

Find the derivative of f’(x)

Solve the integral of √1 + (f’(x))2 dx

Some simple examples to begin with:

arc length constant

Example: Find the length of f(x) = 2 between x=2 and x=3

f(x) is just a horizontal line, so its derivative is f’(x) = 0

Start with:

S =

Put in f’(x) = 0:

S =

S = 3 − 2 = 1

So the arc length between 2 and 3 is 1. Well of course it is, but it's nice that we came up with the right answer!

Interesting point: the "(1 + ...)" part of the Arc Length Formula guarantees we get at least the distance between x values, such as this case where f’(x) is zero.
