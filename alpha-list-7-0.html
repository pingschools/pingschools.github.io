Further explanation

The core reaction is a reaction that causes changes in the core structure.

In the core reaction, the number of atomic numbers and mass numbers of the reactants is the same as the atomic number and mass number of the product

Radioactivity is the process of unstable isotopes towards stable isotopes by decay, by emitting certain particles, among others

alpha α particles beta β ₋₁e⁰ particles gamma particles γ positron particles ₁e⁰

Isotopes are atoms that have the same number of protons but have different numbers of neutrons

The Atomic Number (Z) indicates the number of protons in an atom of an element. If the atom is neutral then the number of protons will be equal to the number of electrons. So the atomic number can also indicate the number of electrons.

 So atomic number = number of protons = number of electrons

Mass Number (A) is the sum of protons and neutrons

Mass Number (A) = Number of protons + Number of Neutrons

Alpha α particles are associated with the Helium nucleus

Helium has 9 isotopes but the most stable is Helium-3 and Helium-4

From question above, An alpha particle has 2 protons, 2 neutrons, and 0 electrons

A helium atom has atomic number 2 (proton = electron = 2) so if there are 0 electrons then the possibility is that Helium atom releases 2 electrons, while the mass number is 4 because there are 2 neutrons

So these alpha particles are related to Helium-4
